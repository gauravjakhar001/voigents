# Voigents Backend

Backend API for Voigents platform.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Start production server:
```bash
npm start
```

## Project Structure

- `src/` - Source code
- `routes/` - API routes
- `controllers/` - Request handlers
- `models/` - Data models
- `middleware/` - Custom middleware
- `config/` - Configuration files
