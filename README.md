# Voigents Project

Full-stack application with separate frontend and backend folders.

## Project Structure

```
voigents/
├── frontend/          # React frontend application
│   ├── src/
│   ├── public/
│   └── package.json
├── backend/           # Backend API
│   ├── src/
│   ├── routes/
│   ├── controllers/
│   ├── models/
│   └── package.json
└── package.json       # Root package.json (optional)
```

## Getting Started

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
npm run dev
```

## Development

- Frontend runs on: `http://localhost:5173` (Vite default)
- Backend runs on: `http://localhost:3000` (configure as needed)
